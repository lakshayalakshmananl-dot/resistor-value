# 🔬 Resistor Color Code Virtual Lab

## 📌 Project Overview

This project is a **Virtual Electronics Lab experiment** that calculates the resistance value of a resistor using its **color bands**.

The user selects the resistor band colors, and the application automatically calculates the **resistance value and tolerance**.

This project is built using **Python and Streamlit** and is designed for **electronics students to learn resistor color coding interactively**.

---

## 🎯 Objective

To determine the **resistance value of a resistor** using the **resistor color code method** through a virtual lab interface.

---

## ⚙️ Features

* Interactive **resistor color band selection**
* Automatic **resistance calculation**
* Displays **formatted resistance values (Ω, kΩ, MΩ)**
* Includes **Theory, Procedure, and Experiment sections**
* Simple and clean **user interface**

---

## 🧪 Experiment Procedure

1. Open the virtual lab.
2. Go to the **Experiment section**.
3. Select the **color bands** of the resistor.
4. Click **Calculate Resistance**.
5. The resistance value will be displayed instantly.

---

## 🧠 Theory

Resistors are passive electronic components used to **limit or regulate current** in electrical circuits.

The resistance value of a resistor is represented by **colored bands** printed on its body.

A typical **4-band resistor** includes:

* **Band 1:** First digit
* **Band 2:** Second digit
* **Band 3:** Multiplier
* **Band 4:** Tolerance

The resistance is calculated using the formula:

Resistance = (Digit1 Digit2) × Multiplier
